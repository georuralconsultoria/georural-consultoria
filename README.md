# GeoRural Consultoria

Versão estática preparada para publicação gratuita no GitHub Pages.

## Publicar

1. Crie um repositório público chamado `georural-consultoria`.
2. Envie para a raiz do repositório: `index.html`, `style.css`, `script.js`, `.nojekyll` e a pasta `assets`.
3. Abra **Settings > Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Selecione a branch **main**, a pasta **/(root)** e clique em **Save**.
6. Aguarde a publicação.

O endereço normalmente será:

`https://SEU-USUARIO.github.io/georural-consultoria/`

## WhatsApp

Abra `script.js` e procure:

```js
const numeroWhatsApp = "";
```

Coloque o número com código do país e DDD, somente números. Exemplo:

```js
const numeroWhatsApp = "5575999999999";
```

## Importante

O GitHub Pages publica sites estáticos. Esta versão não usa PHP e não precisa de XAMPP.
